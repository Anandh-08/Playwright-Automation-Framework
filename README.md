### 🚀 Overview
A robust **End-to-End (E2E) Test Automation Framework** built with **Playwright** and **Java**. This repository contains a scalable testing solution designed for high-speed execution and cross-browser reliability.

---

### 🛠️ Tech Stack & Environment
* **Automation:** Playwright (Java)
* **Build Tool:** Maven
* **IDE:** Eclipse 2025-09 (4.37.0)
* **OS:** macOS (Apple Silicon M2)
* **Reporting:** Playwright Test Reports

---

### ✨ Key Features
* 🏎️ **High Speed:** Native Playwright execution for rapid feedback.
* 🌐 **Cross-Browser:** Automated testing across Chromium, Firefox, and WebKit.
* 🏗️ **Design Pattern:** Implemented using **Page Object Model (POM)** for better maintainability.
* 📊 **Auto-Waiting:** Eliminates flakiness by waiting for elements to be actionable.
